## インストール

<TabsContent value="cli">

<Steps>

<Step>次のコマンドを実行:</Step>

```bash
npx shadcn-ui@latest add sonner
```

<Step>Toasterコンポーネントを追加</Step>

```tsx title="app/layout.tsx" {1,9}
import { Toaster } from "@/components/ui/sonner"

export default function RootLayout({ children }) {
  return (
    <html lang="en">
      <head />
      <body>
        <main>{children}</main>
        <Toaster />
      </body>
    </html>
  )
}
```

</Steps>

</TabsContent>
