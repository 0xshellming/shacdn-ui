## 使用法

```tsx
import { Calendar } from "@/components/ui/calendar"
```

```tsx
const [date, setDate] = React.useState<Date | undefined>(new Date())

return (
  <Calendar
    mode="single"
    selected={date}
    onSelect={setDate}
    className="rounded-md border"
  />
)
```

詳細については [React DayPicker](https://react-day-picker.js.org) ドキュメントを参照してください。

## DayPicker

`<Calendar>` コンポーネントを使用して日付選択機能を構築できます。詳細については [DayPicker](/docs/components/date-picker) ページを参照してください。
