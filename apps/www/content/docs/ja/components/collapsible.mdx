## 使用法

```tsx
import {
  Collapsible,
  CollapsibleContent,
  CollapsibleTrigger,
} from "@/components/ui/collapsible"
```

```tsx
<Collapsible>
  <CollapsibleTrigger>プロジェクトで使用できますか？</CollapsibleTrigger>
  <CollapsibleContent>
    はい。個人的および商用プロジェクトで自由に使用できます。アトリビューションは不要です。
  </CollapsibleContent>
</Collapsible>
```
