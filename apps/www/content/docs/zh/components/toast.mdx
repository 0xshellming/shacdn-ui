## 使用

`useToast` hook 返回一个 `toast` 函数，你可以使用它来显示一个 toast。

```tsx
import { useToast } from "@/components/ui/use-toast"
```

```tsx {2,7-10}
export const ToastDemo = () => {
  const { toast } = useToast()

  return (
    <Button
      onClick={() => {
        toast({
          title: "Scheduled: Catch up",
          description: "Friday, February 10, 2023 at 5:57 PM",
        })
      }}
    >
      Show Toast
    </Button>
  )
}
```

<Callout>

要同时显示多个 toast，你可以更新 `use-toast.tsx` 中的 `TOAST_LIMIT`。

</Callout>
