## 用法

```tsx

  Collapsible,
  CollapsibleContent,
  CollapsibleTrigger,
} from "@/components/ui/collapsible"
```

```tsx

  <CollapsibleTrigger>我可以在我的项目中使用这个吗？</CollapsibleTrigger>
  <CollapsibleContent>
    是的。免费用于个人和商业项目。无需注明来源。
  </CollapsibleContent>
</Collapsible>
```
