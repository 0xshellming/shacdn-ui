```tsx

```

```tsx
<ScrollArea className="h-[200px] w-[350px] rounded-md border p-4">
滑稽小丑半夜偷偷潜入城堡，到处留下了笑话：放在国王的枕头下、放在他的汤里，甚至放在王室马桶里。国王勃然大怒，但他似乎无法阻止滑稽小丑。然后，有一天，王国的人们发现滑稽小丑留下的笑话非常有趣，他们禁不住哈哈大笑。而一旦他们开始笑，就再也停不下来了。
</ScrollArea>
```
