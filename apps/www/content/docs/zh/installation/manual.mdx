## 步骤

### 添加 Tailwind CSS

组件是用 Tailwind CSS 进行样式化的。您需要在您的项目中安装 Tailwind CSS。

[请按照 Tailwind CSS 安装说明来开始。](https://tailwindcss.com/docs/installation)

### 添加依赖项

向您的项目中添加以下依赖项：

```bash
npm install tailwindcss-animate class-variance-authority clsx tailwind-merge
```

### 添加图标库

如果您正在使用“默认”样式，请安装 `lucide-react`：

```bash
npm install lucide-react
```

如果您正在使用“纽约”样式，请安装 `@radix-ui/react-icons`：

```bash
npm install @radix-ui/react-icons
```

### 配置路径别名

我使用 `@` 别名。这是我在 tsconfig.json 中配置它的方式：

```json {3-6} title="tsconfig.json"
{
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@/*": ["./*"]
    }
  }
}
```

`@` 别名是一种偏好。如果您愿意，可以使用其他别名。

**如果您使用其他别名（比如 ~），则在添加组件时需要更新导入语句。**

### 添加 cn 助手

我使用一个 `cn` 助手来更轻松地有条件地添加 Tailwind CSS 类。以下是我在 `lib/utils.ts` 中定义它的方式：

```ts title="lib/utils.ts"
import { clsx, type ClassValue } from "clsx"
import { twMerge } from "tailwind-merge"

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs))
}
```
